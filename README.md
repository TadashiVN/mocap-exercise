# mocap-exercise
A Wearable Inertial Measurement Unit-Based Motion Capture System with Deep Learning for Exercise Posture Analysis

- This is the first version of project, just check the algorithm of MPU and how to AI checking firstly !

## Overview of this project (Final version)

<img width="1190" height="597" alt="image" src="https://github.com/user-attachments/assets/9530847c-84c4-4763-82b6-90a8fcdb93fe" />

- Hardware:  ESP32 x 6; MPU6050 x 5; Pin Lithium x 5-6
- Software: Arduino IDE, Pycharm.

## Description
- ESP32 + MPU6050 ---Serial Port [Data (pitch + roll + yaw)] ---> PC/Laptop ---AI model-->Evaluate pose/exercise
