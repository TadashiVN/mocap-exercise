# Images

This folder stores diagrams, photos, and screenshots used in the README.

Recommended files:

- `system_architecture.svg`: included system diagram.
- `hardware_setup.jpg`: real ESP32 + MPU6050 hardware setup.
- `wearable_setup.jpg`: optional photo showing sensor placement on the body.
- `serial_monitor.png`: Arduino Serial Monitor showing IMU CSV data.
- `web_demo.png`: Flask web demo showing prediction result.
- `data_plot.png`: optional IMU signal plot.

See `IMAGE_GUIDE.md` for suggested captions and what each image should show.
